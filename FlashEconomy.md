FlashEconomy is the server's currency and banking system. You earn money by playing, spend it on boosters and coinflips, and stash it in the bank to earn interest.

---

## Starting Balance

New players start with **$2,500**.

---

## Earning Money

| Source | Payout |
|--------|--------|
| Mining coal | $1.50 per block |
| Mining copper | $2.00 per block |
| Mining iron | $2.00 per block |
| Mining lapis | $2.50 per block |
| Mining redstone | $2.50 per block |
| Mining nether quartz | $2.50 per block |
| Mining nether gold | $3.00 per block |
| Mining gold | $3.00 per block |
| Mining diamond | $5.00 per block |
| Mining emerald | $6.00 per block |
| Killing mobs | Varies by mob health |
| Active booster | All earnings doubled |

## Commands

### Wallet

| Command | Description |
|---------|-------------|
| `/balance` | Check your current wallet balance |
| `/bal` | Alias for `/balance` |
| `/baltop` | View the top 10 richest players and your own ranking |
| `/pay <player> <amount>` | Send money to another player |
| `/paytoggle` | Toggle whether other players can send you money |

### Bank

| Command | Description |
|---------|-------------|
| `/bank` | Opens the banking interface |
| `/bank balance` | Check your bank balance and storage capacity |
| `/bank interest` | View your interest rate and projected daily/weekly/monthly earnings |
| `/bank stocks` | View available stocks on the market |
| `/bank loan` | Check your active loans and any overdue status |
| `/bank portfolio` | View your current stock investments |

The bank has 5 levels, each with higher interest rates and storage capacity. Late loan payments cost $500/day.

### Boosters

| Command | Description |
|---------|-------------|
| `/purchasebooster` | Buy a 30-minute money booster for $15,000 - doubles all earnings |
| `/boostertime` | Check remaining time on your active booster |

### Coinflip

| Command | Description |
|---------|-------------|
| `/coinflip create <amount> [HEADS\|TAILS]` | Create a wager - money is withdrawn immediately |
| `/coinflip join <id>` | Join an existing coinflip |
| `/coinflip list` | Browse all active coinflips in a GUI |
| `/coinflip cancel` | Cancel your active coinflip and get your money back |
