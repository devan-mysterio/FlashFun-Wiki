FlashEssentials is the core utility plugin that adds quality-of-life commands for players on Flash-Networks.

---

## Communication

| Command | Description |
|---------|-------------|
| `/msg <player> <message>` | Send a private message to another player |
| `/reply <message>` | Reply to the last player who messaged you |
| `/discord` | Get the link to the Flash-Networks Discord server |
| `/afk` | Mark yourself as AFK - toggles on and off |

---

## Homes

| Command | Description |
|---------|-------------|
| `/homes` | Open your homes menu to set, teleport to, or delete homes |

---

## Teleportation

| Command | Description |
|---------|-------------|
| `/tpa <player>` | Send a teleport request to another player (30-second timeout) |
| `/tpatoggle` | Toggle whether players can send you TPA requests |
| `/rtp` | Teleport to a random location 250-1000 blocks away - costs $250 |
| `/back` | Teleport to your last location |

RTP only works in the main world and finds a safe landing spot. It does not drop you into water, lava, or dangerous biomes.

---

## Inventory Utilities

| Command | Description |
|---------|-------------|
| `/craft` | Open a crafting table anywhere |
| `/enderchest` | Open your ender chest anywhere |
| `/disposal` | Open a trash slot to delete unwanted items |
| `/hat` | Wear the item you're holding as a hat |

---

## Player Info

| Command | Description |
|---------|-------------|
| `/stats [player]` | View your (or another player's) statistics in a GUI |
| `/onlinelist` | See all currently online players |
| `/stafflist` | See which staff members are currently online |
| `/rules` | Read the server rules |
| `/buy` | Open the server store to view ranks and supporter perks |

---

## Social & Fun

| Command | Description |
|---------|-------------|
| `/pvptoggle` | Toggle PvP on or off for yourself |
| `/nickname <name>` | Set a custom display nickname |
| `/togglescoreboard` | Show or hide the sidebar scoreboard |
| `/suicide` | End it all |
