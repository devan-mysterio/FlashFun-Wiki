The Multitool is a [Technical Gadget](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Technical-Gadgets) added in FlashFun.<br>
A single handheld item that consolidates multiple magical and technical tools into one. Cycle between modes with sneak + right-click, then right-click to use.

## Stats

|-|-|-|-|
| T1 | 22 | 20 J | Magic Eye, Wind, Grapple, Portable Craft |

| T5 | 32 | 60 J | Magic Eye, Wind, Grapple, Portable Craft |
| T6 | 36 | 75 J | Magic Eye, Wind, Grapple, Portable Craft |
| T7 | 42 | 100 J | All modes |

## Modes

| Mode | Tiers | Charge Cost | Effect |
|-|-|-|-|
| Magic Eye | T1–T7 | 3 J | Shoot ender pearl (requires full Ender Armor) |
| Wind | T1–T7 | 2 J | Launch yourself forward in your look direction |
| Grapple | T1–T7 | 1 J | Cast a grappling hook; right-click again to pull yourself |
| Portable Craft | T1–T7 | - | Open a 3x3 crafting grid |
| Gold Pan | T7 only | - | Use gold pan functionality |
| Nether Gold Pan | T7 only | - | Use nether gold pan functionality |
| Water | T7 only | 2 J | Extinguish yourself when on fire |
| Storm | T7 only | 5 J | Strike lightning at your crosshair (10 block range) |

## Usage

- **Sneak + right-click** to cycle to the next mode (shown in action bar)
- **Right-click** to use the current mode
