The Hologram Projector is a [Technical Gadget](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Technical-Gadgets) added in FlashFun.<br>
A placeable item that displays a floating holographic text label above its position.

## Hologram Projector

- **Yield:** 3 per craft
- **Unlock Level:** 20
- **Max text length:** 32 characters

## Usage

Place the Hologram Projector like a block. A hologram appears above it displaying text.
Right-click the placed projector to edit the displayed text (up to 32 characters).
