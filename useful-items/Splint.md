The Splint is a [Useful Item](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Useful-Items) added in FlashFun.<br>
A simple bone-support brace made from sticks and string. Restores a small amount of health.



## Splint

- **Healing:** Restores 2 hearts
- **Usage:** Right-click to use
- **Stack Size:** 4
- **Unlock Level:** 1

### Recipe

| | | |
|-|-|-|
| - | Stick | - |
| String | String | String |
| - | Stick | - |

**Output:** 4 Splints
