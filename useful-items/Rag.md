The Rag is a [Useful Item](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Useful-Items) added in FlashFun.<br>
A basic medical supply made from cloth. Restores a small amount of health and puts out fires.

## Rag

- **Healing:** Restores 2 hearts
- **Extinguishes fire** on use
- **Usage:** Right-click to use
- **Stack Size:** 4
- **Unlock Level:** 1

### Recipe

| | | |
|-|-|-|
| Cloth | Cloth | Cloth |
| String | - | String |
| Cloth | Cloth | Cloth |

**Output:** 2 Rags



*A glorified piece of cloth. Still better than nothing.*
