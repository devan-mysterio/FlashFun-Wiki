The Portable Crafter is a [Useful Item](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Useful-Items) added in FlashFun.<br>
A handheld crafting solution - a book infused with a crafting table so you can craft anywhere.



## Portable Crafter

- **Effect:** Opens a crafting table GUI on right-click
- **Usage:** Right-click to open
- **Stack Size:** 1
- **Unlock Level:** 5

### Recipe

Crafted shapeless using the following items:

- Book
- Crafting Table

**Output:** 1 Portable Crafter



*A book that IS a crafting table. The engineers are getting too clever.*
