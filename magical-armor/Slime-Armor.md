Slime Armor is a [Magical Armor](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Magical-Armor) set added in FlashFun.<br>
Gelatinous green leather armor coated in sticky slime residue. The full set makes you bounce, move faster, and land without injury.

Slime and Reinforced Slime pieces are **interchangeable** for set bonus purposes - mixing the two sets still activates the full set bonus.

## Set Bonus (Full Set or mixed with Reinforced Slime)

- **Jump Boost II**
- **Speed I**
- **No fall damage**
