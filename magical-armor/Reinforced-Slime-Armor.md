Reinforced Slime Armor is a [Magical Armor](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Magical-Armor) set added in FlashFun.<br>
Slime-infused iron armor with a hardened gel core. Provides stronger base protection than the original Slime set and includes Protection IV and Unbreaking IV enchantments.

Reinforced Slime and regular Slime pieces are **interchangeable** for set bonus purposes - mixing the two sets still activates the full set bonus.

## Set Bonus (Full Set or mixed with Slime Armor)

- **Jump Boost II**
- **Speed I**
- **No fall damage**

## Pieces

All pieces have: **Protection II**, **Unbreaking IV**
