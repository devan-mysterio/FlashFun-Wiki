Minion Skulls are [Miscellaneous](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Miscellaneous) items added in FlashFun.<br>
Each skull is a crafting ingredient used to build its corresponding [Minion](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Minions). Craft the skull first, then use it in the minion's recipe.

All skulls share the same recipe shape - only the dye color changes.

## Recipe (All Skulls)

| | | |
|-|-|-|
| Dye | Resin Clump | Dye |
| Resin Clump | Infernal Essence | Resin Clump |
| Dye | Resin Clump | Dye |

**Unlock Level:** 10 (all skulls)
