The Cargo Manager is part of [Cargo Management](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Cargo-Management) in FlashFun. It is the central hub of a cargo network - every network requires exactly one.

## Cargo Manager

- **Unlock Level:** 25
- **Yield:** 1 per craft

## Usage

Place the Cargo Manager as a block. Right-click to view network status, connected nodes, and item flow statistics. All cargo nodes within 4 blocks are automatically linked to this manager.

One Cargo Manager per network. The manager rescans its network every 5 seconds and processes item transfers every second.
