FlashSettings is a per-player preferences menu accessible via `/settings`. All toggles are saved per player and persist across sessions.

---

## Command

| Command | Description |
|---------|-------------|
| `/settings` | Opens your personal settings menu |

---

## Page 1 - General

| Setting | Description |
|---------|-------------|
| Receive Chat | Toggle public chat messages on or off |
| Private Messages | Toggle whether other players can send you private messages |
| Scoreboard Display | Show or hide the sidebar scoreboard |
| Death Messages | Toggle death messages appearing in chat |
| Payment Alerts | Toggle notifications when you receive money (you still get the money either way) |
| Payment System | Toggle whether other players can send you money at all |
| Hotbar Messages | Toggle hotbar notification messages |
| Server Messages | Toggle server announcements and broadcast messages |
| PvP Mode | Toggle your PvP status on or off |
| AFK Status | Toggle automatic AFK detection |
| Allow Mentions | Toggle whether players can @mention you in chat |
| Teleport Requests | Toggle whether players can send you TPA requests |
| Daily Income Alert | Toggle daily income reminder notifications |
| Ultimate Steak Drop | Toggle notifications when an Ultimate Steak drops |

---

## Page 2 - Visual

| Setting | Description |
|---------|-------------|
