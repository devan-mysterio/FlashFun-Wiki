Castor is the Flash-Networks Discord bot. It handles the server economy, music, support tickets, community features, and Minecraft integration all in one place.

---

## Economy

| Command | Description |
|---------|-------------|
| `/balance` | Check your current coin balance |
| `/daily` | Claim your daily coins - streak multiplier increases with consecutive days |
| `/baltop` | View the top 10 richest members |
| `/pay <user> <amount>` | Send coins to another member |

---

## Gambling

| Command | Description |
|---------|-------------|
| `/roulette <bet> <amount>` | Bet on red/black, odd/even, high/low, or a specific number (0-36) |
| `/slots <amount>` | Spin the slot machine |
| `/coinflip <user> <amount>` | Challenge another member to a heads/tails bet - they can Accept or Decline |
| `/fish` | Fish for coins (10-second cooldown) |
| `/steal <user>` | Attempt to steal coins from another member (30-second cooldown, success not guaranteed) |

---

## Loot

| Command | Description |
|---------|-------------|

## Profile

| Command | Description |

## Music

Music commands work in the designated music channel. You must be in a voice channel to use them.

| Command | Description |
|---------|-------------|
| `/join` | Bot joins your voice channel |
| `/play <song>` | Play a song by name or URL |
| `/pause` | Pause or resume playback |
| `/skip` | Skip the current track |
| `/stop` | Stop playback and clear the queue |
| `/restart` | Restart the current track |
| `/repeat` | Toggle repeat mode |
| `/shuffle` | Shuffle the queue |
| `/queue` | View the next 20 tracks in queue |
| `/volume <1-100>` | Set the playback volume |
| `/nowplaying` | Show what's currently playing |

## Support & Community

- **Tickets** - Click the Open Ticket button in the support channel to create a private support ticket
- **Staff Applications** - Click Apply for Staff in the applications channel to submit a staff application
- **Suggestions** - Use the suggestions channel to submit ideas for Castor, the Minecraft server, or the Discord
