Circuit Boards are [Technical Components](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Technical-Components) added in FlashFun.<br>
High-tier crafting components used in advanced machines and automation devices.

## Basic Circuit Board

- **Yield:** 4 per craft
- **Used in:** Advanced Circuit Board, various machines
- **Unlock Level:** 20

### Recipe

| | | |
|-|-|-|
| Steel Plate | Power Crystal | Steel Plate |
| Heating Coil | Redstone Block | Heating Coil |
| Steel Plate | Power Crystal | Steel Plate |

## Advanced Circuit Board

- **Used in:** High-tier machines and gadgets
- **Unlock Level:** 22

### Recipe

| | | |
|-|-|-|
| Lapis Block | Lapis Block | Lapis Block |
| Redstone Block | Basic Circuit Board | Redstone Block |
| Lapis Block | Lapis Block | Lapis Block |

*The difference between basic and advanced is three rows of lapis. And a lot of prerequisites.*
