FlashKits provides starter kits and daily login rewards for players.

---

## Command

| Command | Description |
|---------|-------------|
| `/kit` | Opens the kits menu |

---

## Daily Login Reward

Log in once every 24 hours to automatically receive coins.

| Rank | Daily Reward |
|------|-------------|
| Default | $1,000 |
| Illuminate | $2,000 |

*Two kits, one daily allowance, and a shovel for claiming land. Typical new player experience.*
