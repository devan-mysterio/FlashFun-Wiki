The Crafting Altar is a [Basic Machine](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Basic-Machines) added in FlashFun.<br>
The primary crafting station for all FlashFun items. Place it in the world and right-click to access the custom crafting interface.

## Crafting Altar

- **Effect:** Opens the FlashFun crafting GUI on right-click
- **Placement:** Place in world like a normal block - right-click to use
- **Fire resistant** - won't burn
- **Unlock Level:** 10
- **Crafted at:** Vanilla Crafting Table (not the Altar itself)

### Recipe

| | | |
|-|-|-|
| Amethyst Block | Nether Star | Amethyst Block |
| Nether Star | Crafting Table | Nether Star |
| Amethyst Block | Nether Star | Amethyst Block |

*This is the only FlashFun item crafted at a vanilla Crafting Table.*

*A crafting table made of amethyst and nether stars. Completely normal construction materials.*
