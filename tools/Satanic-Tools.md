Satanic Tools are a set of [Tools](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Tools) added in FlashFun.<br>
Netherite tools infused with demonic energy. All four are unbreakable, fire resistant, and crafted using the same shapeless recipe - replace the netherite tool with the appropriate type.

All four are crafted **shapeless** using: Demonic Token + Infernal Essence + the corresponding Netherite tool.

## Satanic Pickaxe

- **Attack Damage:** +7.0
- **Attack Speed:** -2.8
- **Mining Efficiency:** +7.0
- **Submerged Mining Speed:** +2.0
- **Enchantments:** Efficiency VII, Fortune V
- **Unbreakable**, Fire resistant
- **Unlock Level:** 16

## Satanic Axe

- **Attack Damage:** +11.0
- **Attack Speed:** -3.0
- **Enchantments:** Sharpness X, Silk Touch I, Efficiency V, Fire Aspect III
- **Unbreakable**, Fire resistant
- **Unlock Level:** 16

## Satanic Shovel

- **Attack Damage:** +6.5
- **Attack Speed:** -3.0
- **Enchantments:** Efficiency X, Fortune V
- **Unbreakable**, Fire resistant
- **Unlock Level:** 16

## Satanic Hoe

- **Attack Damage:** +2.0
- **Attack Speed:** -1.0
- **Enchantments:** Efficiency X, Fortune V
- **Unbreakable**, Fire resistant
- **Unlock Level:** 16

*Forged in hellfire. All four of them. Someone down there has a full tool shed.*
