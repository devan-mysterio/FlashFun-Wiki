Explosive Tools are a pair of [Tools](https://github.com/devan-mysterio/FlashFun-Wiki/wiki/Tools) added in FlashFun.<br>
TNT-powered tools that break a 3x3 area in a single swing. Both respect protected regions and only affect their respective block types.

## Explosive Pickaxe

- **Effect:** Mines a 3x3 area of pickaxe-mineable blocks per swing
- **Respects** protected areas
- **Unlock Level:** 22

### Recipe

| | | |
|-|-|-|
| TNT | Synthetic Diamond | TNT |
| - | Ferrosilicon | - |
| - | Ferrosilicon | - |

## Explosive Shovel

- **Effect:** Digs a 3x3 area of shovel-mineable blocks per swing
- **Respects** protected areas
- **Unlock Level:** 22

### Recipe

| | | |
|-|-|-|
| - | Synthetic Diamond | - |
| - | TNT | - |
| - | Ferrosilicon | - |

*Why mine one block when you can mine nine? Why dig one hole when you can dig nine? The math checks out.*
